# AgentMesh

<a href="/README.md">English</a> | 中文

AgentMesh是一个 **多智能体 (Multi-agent) 平台** ，提供AI Agent开发框架、多Agent间的通信协议、复杂任务规划和自主决策。 基于该平台可以快速构建你的Agent Team，通过Agents之间的协作完成任务。
